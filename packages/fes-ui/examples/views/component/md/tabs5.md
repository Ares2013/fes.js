| 属性           | 说明                       | 类型     |        默认值                                          |
|:--------------|:--------------------------|:--------|:-----------------------------------------------------|
| key          |  用于标识当前选项卡 | String  |        null              |
| label          | tab头部显示的文案 | String   |                     null                        |
| icon        | 控制tab头部显示的icon | String   |                     null                        |
| disabled          | 是否禁用该选项卡 | Boolean   |                     false                        |
