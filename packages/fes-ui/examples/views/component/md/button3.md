| 属性           | 说明                       | 类型     |        默认值                                          |
|:--------------|:--------------------------|:--------|:-----------------------------------------------------|
| type          | 按钮的类型，可选值是primary、ghost、text、info、success、warn、error或者无  | String  |        null              |
| long          | 控制按钮是否显示为100%宽度 | Boolean   |                     false                        |
| circle        | 控制按钮显示显示为圆角 | Boolean   |                     false                        |
| icon          | 控制按钮内部左侧显示的小图标 | Boolean   |                     false                        |
