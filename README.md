# Fes.js

Fes.js 是一个管理台应用解决方案，提供初始项目、开发调试、编译打包的命令行工具，内置布局、权限、数据字典、状态管理、Api等多个模块，文件目录结构即路由，用户只需要编写页面内容。基于Vue.js，内置管理台常用能力，让用户写的更少，更简单。经过多个项目中打磨，趋于稳定。

## 使用

```bash
# install
npm install @webank/fes-cli -g

# create a project
fes init [project]

cd [project] && npm i

# start dev
npm run dev

# build
npm  run build
```
